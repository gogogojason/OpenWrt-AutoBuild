# OpenWrt-AutoBuild

#### 基于OpenWrt-Actions-Lean+Lienol-自动化在线编译脚本  
在仓库Settings->Secrets中分别添加 PPPOE_USERNAME, PPPOE_PASSWD 可设置默认拨号账号密码.有 安全隐患

Secrets中添加 SCKEY 可通过Server酱 推送编译结果到微信
